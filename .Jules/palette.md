# Palette Journal

## 2024-05-22 - Window Control Patterns
**Learning:** Users instinctively expect macOS-style traffic light buttons to interact on hover (showing X, -, +) and support keyboard navigation. Missing these micro-interactions makes the UI feel "dead".
**Action:** When implementing OS-mimicking UIs, ensure standard behaviors (hover states, aria-labels) are replicated, not just the visual look.
